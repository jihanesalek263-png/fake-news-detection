# Fake News Detection using Multilingual Transformers

## Overview

This project presents a multilingual Fake News Detection system based on the XLM-RoBERTa Transformer model.

The project covers the complete machine learning pipeline, including data exploration, preprocessing, baseline experiments, model fine-tuning, and evaluation on multiple benchmark datasets.

## Datasets

The experiments were conducted using three public datasets:

- LIAR
- FakeNewsNet
- Arabic Fake News Dataset (AFND)

## Project Structure

| Notebook | Description |
|----------|-------------|
| 01_Data_Exploration.ipynb | Exploratory Data Analysis (EDA) |
| 02_LIAR_Audit_Baselines.ipynb | Baseline models on the LIAR dataset |
| 03_AFND_Audit_Baselines.ipynb | Baseline models on the AFND dataset |
| 04_FakeNewsNet_Audit_Baselines.ipynb | Baseline models on the FakeNewsNet dataset |
| 05_XLMR_Multilingual_Training.ipynb | Fine-tuning and evaluation of the XLM-RoBERTa model |
